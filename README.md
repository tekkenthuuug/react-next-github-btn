# react-next-github-btn

## Installation

```sh
npm install react-next-github-btn
# OR
yarn add react-next-github-btn
```

## Usage

```jsx
import GitHubButton from 'react-next-github-btn';

<GitHubButton href='https://github.com/ntkme'>Follow @ntkme</GitHubButton>;
```
